# Vue 2.6 + Pinia Demo
[netlify备用](https://main--fancy-froyo-08882c.netlify.app/)

这是一个使用 Vue 2.6 和 Pinia 的简单示例，旨在帮助开发者了解如何在 Vue 2.6 中使用 Pinia 状态管理库。

## 功能

该示例包含以下功能：

- 显示计数器值
- 点击按钮增加计数器值
- 点击按钮减少计数器值
- 使用 Pinia 管理计数器状态
- 使用 Pinia 持久化计数器状态

## 技术栈

- Vue 2.6
- Pinia 2.0

## 如何运行

1. 安装依赖

   ```bash
   npm install
   ```

2. 运行示例

   ```bash
   npm run serve
   ```
